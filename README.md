# Portfolio

Dự án portfolio được xây dựng bằng React và sử dụng Tailwind CSS làm thư viện CSS.

## Yêu cầu

-   [Node.js](https://nodejs.org/en/) >= 10.15.3

## Cài đặt

```sh
cd porfolio
npm install
```

## Chạy dự án

```sh
npm run dev
```

## Deploy

Thay đổi biến `homepage` trong file `package.json` thành tên domain của bạn.

```sh
npm run deploy
```
