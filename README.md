# Tên dự án

Mô tả ngắn gọn về dự án của bạn.

## Yêu cầu

-   [Node.js](https://nodejs.org/en/) >= 10.15.3

## Cài đặt

```sh
cd porfolio
npm install
```

## Chạy dự án

```sh
npm run dev
```

## Deploy

Thay đổi biến `homepage` trong file `package.json` thành tên domain của bạn.

```sh
npm run deploy
```
